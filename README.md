### Решение 1. 

*Кеширование - это временное хранение в быстродоступной памяти, призванное сократить время доступа к этим данным. Кеширование позволяет решить много проблем в работе основной базы данных. Такие как снижение и распределение высокой нагрузки на базу данных в период чрезмерно высокого количестра запросов. Решение проблем с низкой произкой базы данных за счет размещения часто запрашиваемых данных в быстрой памяти. Может помочь масштабировать базе данных без значительных затрат на масштабирование оснойной базы данных. Кеширование минимизирует конфликты чтения/записи храня актуальные данные. Помогает географически распределить базы данных снижая задержки и время отклика при работе с базой. Благодаря увеличению скорости отклика улучшается пользовательский опыт, так как пользователи быстрее получают данные к которым обращались*
